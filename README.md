# MotherLand-Tours - Vitrine Website

Spécialiste du voyage authentique et culturel au Bénin.

## Features
- Interactive 7/10 days Itinerary
- Dynamic Pricing for Paris, Abidjan, Morocco, and Canada
- Premium Smooth & Rounded UI
- Fully Responsive Design
- WhatsApp & Direct Email Integration

## Deployment on Render
This project is configured for [Render](https://render.com).
- Uses PostgreSQL database.
- Auto-deploys via `render.yaml` Blueprint.
